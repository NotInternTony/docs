## Monitored Services

Grain continuously monitors the following core services:

### API

The Grain REST API that powers payment creation, refunds, webhooks, and all programmatic interactions. This is the service your integration communicates with directly.

### Dashboard

The web-based merchant dashboard used for managing settings, viewing transactions, generating reports, and accessing support.

### Website

The public-facing Grain website. This covers availability and performance of the main Grain site.

<Tip>
  If the API is operational but you are experiencing issues, the problem is likely related to your integration configuration. Please contact us to investigate any issues.
</Tip>
