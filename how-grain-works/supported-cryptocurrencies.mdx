## Supported Cryptocurrencies

Your customers can pay using any of the following crypto assets. Grain automatically converts the received crypto to your settlement currency regardless of which asset is used.

| Asset   | Name          | Type        |
|---------|---------------|-------------|
| USDC    | USD Coin      | Stablecoin  |
| USDT    | Tether        | Stablecoin  |
| DAI     | Dai           | Stablecoin  |
| ETH     | Ether         | Native coin |
| BTC     | Bitcoin       | Native coin |
| SOL     | Solana        | Native coin |
| MATIC   | Polygon       | Native coin |
| AVAX    | Avalanche     | Native coin |

<Tip>
  Stablecoins like USDC and USDT are pegged to the US Dollar, which means minimal price fluctuation between the time your customer pays and when Grain converts the funds. Many merchants see the majority of their crypto payments in stablecoins.
</Tip>

## Supported Networks

Grain supports payments across multiple blockchain networks. When a customer initiates a payment, Grain provides deposit addresses for each supported network so the customer can choose the one that works best for their wallet.

| Network     | Native Asset | Supported Tokens    | Avg. Confirmation Time |
|-------------|-------------|----------------------|------------------------|
| Ethereum    | ETH         | USDC, USDT, DAI     | ~2 minutes             |
| Solana      | SOL         | USDC, USDT          | ~5 seconds             |
| Polygon     | MATIC       | USDC, USDT, DAI     | ~5 seconds             |
| Base        | ETH         | USDC                | ~2 seconds             |
| Arbitrum    | ETH         | USDC, USDT          | ~1 second              |
| Avalanche   | AVAX        | USDC, USDT          | ~2 seconds             |
| Bitcoin     | BTC         | —                   | ~30 minutes            |

## How Network Detection Works

You do not need to configure or manage blockchain networks. When Grain creates a payment session, it generates deposit addresses for all supported networks automatically. The customer selects their preferred network in the Grain checkout interface, and Grain monitors the correct address accordingly.

If a customer sends funds on an unsupported network or to the wrong address type, Grain's monitoring system flags the transaction and your support team can work with Grain to resolve it.

## Multi-Network Tokens

Some tokens like USDC exist on multiple networks. Grain treats these as equivalent — a USDC payment on Ethereum and a USDC payment on Solana both settle to the same fiat amount. The customer benefits from choosing a network with lower gas fees and faster confirmation times, while you receive the same settlement regardless.
